# Portfolio Website - Kishore T S

A modern, responsive portfolio website showcasing AI/ML projects, skills, and experience.

## Features

- ✨ **Modern Design**: Clean, minimal design with card-based layouts
- 🌓 **Dark/Light Theme**: Toggle between themes with persistent preference
- 📱 **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- 🎨 **Smooth Animations**: Fade-in and scroll animations for better UX
- ⚡ **Fast Loading**: Optimized for performance and SEO
- 🎯 **Recruiter-Friendly**: Clear sections, easy navigation, mobile-friendly

## Sections

1. **Hero Section**: Name, title, tagline, and call-to-action buttons
2. **About Me**: Personal introduction and background
3. **Skills**: Categorized skills (AI/ML, NLP, Data Science, etc.)
4. **Projects**: Showcase of 4 AI/ML projects with GitHub links
5. **Experience**: Work experience and internships
6. **Education**: Academic background
7. **Resume**: Download link for PDF resume
8. **Contact**: Contact information and contact form

## Setup

1. Ensure all files are in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `Resume.pdf`

2. Open `index.html` in a web browser or deploy to a web server.

## Customization

### Update Personal Information
Edit the content in `index.html` to update:
- Name, title, tagline
- About section text
- Skills list
- Projects details
- Experience and education

### Update Links
- GitHub: Update all GitHub links
- LinkedIn: Update LinkedIn profile link
- Email: Update email address in contact section and mailto links

### Theme Colors
Edit CSS variables in `styles.css` under `:root` and `[data-theme="dark"]` to customize colors.
